# Socket.io Real-Time Chat Application

## Overview
This project is a **real-time chat application** built using the **MERN stack** with **Socket.io** for bidirectional communication. Users can join a global chat room, send and receive messages in real-time, see who is online, and view typing indicators.  

**Technologies Used:**
- **Backend:** Node.js, Express.js, Socket.io
- **Frontend:** React.js (Vite)
- **Real-Time Communication:** Socket.io
- **Styling:** Tailwind CSS (optional)
- **Version Control:** Git & GitHub

---

## Features Implemented
1. Real-time messaging in a global chat room
2. Typing indicator for active users
3. Online/offline status updates
4. Multiple users supported simultaneously
5. Clean, responsive UI
